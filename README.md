# Yılan Oyunu

Klasik yılan oyununu Python'da uyarlayan bir projedir.

## Açıklama

Ekranda hareket eden yılan, yem yiyerek uzar. Duvarla veya kendisiyle çarpınca oyun biter. Klasik oyun mekaniklerini içerir ve basit arayüzü vardır.

## Kullanım

1. Python 3'ü kur
2. Gereken kütüphaneyi yükle: `pip install pygame`
3. `YilanOyunu.py` çalıştır
4. Oyuna başla

## Kontroller

- **Yukarı Ok**: Yukarı hareket
- **Aşağı Ok**: Aşağı hareket
- **Sağ Ok**: Sağa hareket
- **Sol Ok**: Sola hareket
